Swarm 相关概念

Stack

Stack 指一个应用需要的一整套容器，例如前端、后端API、BFF等，由多个 service 构成。

Service

一个 Service 指一个 docker compose运行后的一个服务，可以存在多个容器的副本

Image

一个Docker镜像

network

容器分配的IP地址
