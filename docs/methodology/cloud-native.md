云原生包含了一组应用的模式，用于帮助企业快速，持续，可靠，规模化地交付业务软件。云原生由微服务架构，DevOps 和以容器为代表的敏捷基础架构组成。

通俗来说就是整合了微服务、容器化、Devops、持续交付得个概念集合。
