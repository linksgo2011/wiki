HBuilder方案

  • mui
  • Hbuilder

成本低廉，开发方面，原生的webview API和切换性能也可
参考案例：https://github.com/linksgo2011/tinda_app_cordova/

注意事项

  • 每个页面为一个独立的窗口
  • 返回会有数据刷新问题
  • 窗口内如果发生页面跳转，会造成路由不一致
  • 尽量使用页面 + API的方式快速开发
