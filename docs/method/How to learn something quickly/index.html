技术类知识学习的流程

  1. 回答几个问题
    • 这项技术是什么？
    • 这项技术的意义？
    • 这项技术的定位？
    • 为什么需要学习这项技术
  2. 收集入门教程
  3. 阅读入门教程产出笔记或者思维导图
  4. 使用小项目练习
  5. 阅读官方文档了解更多
  6. 搞懂原理（可选）

学习一门新技术的要点

  • 咨询有经验的人，需要提前准备问题，寻求一些好的资料和书籍
  • 阅读速度不宜过快，安排大块时间。step by step
  • 一定要有产出
  • 可以编写一些例子和demo放到wiki中
