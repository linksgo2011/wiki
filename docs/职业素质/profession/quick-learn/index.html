web 工程师知识系统快速学习的一些要点 2019-12-05

学习的一般过程

  • 采访专业的人，了解知识框架和学习路线
  • 倍速看视频教程，了解如何做
  • 编写代码，动手实践，遇到问题搜索
  • 落入知识系统，可以将做的实验作为例子编写笔记
  • 看相关的书深入学习，了解原理以及使用中的一些注意事项
  • 看源码
  • 将知识系统中有价值的内容整理成博客

学习的注意事项

  • 少量学习，多去实践
