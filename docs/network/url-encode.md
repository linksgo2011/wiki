## URL 中需要编码的字符



| 字符 | 转义码 | URL中用途                  |      |
| ---- | ------ | -------------------------- | ---- |
| +    | %2B    | URL 中+号表示空格          |      |
| /    | %2F    | 分隔目录和子目录           |      |
| ?    | %3F    | 分隔实际的 URL 和参数      |      |
| %    | %25    | 指定特殊字符               |      |
| \#   | %23    | 锚点                       |      |
| &    | %26    | URL 中指定的参数间的分隔符 |      |
| =    | %3D    | URL 中指定参数的值         |      |
