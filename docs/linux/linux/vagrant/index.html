虚拟机管理工具

安装步骤

  1. 安装 VisualBox
  2. 安装 Vagrant
  3. 安装 Vagrant 封装好的镜像 http://www.vagrantbox.es/
  4. 启动

常用命令

添加虚拟机

vagrant box add centos7 /Users/apple/Downloads/centos7.box

初始话虚拟机

vagrant init centos7

启动虚拟机

vagrant up

查看当前运行的虚拟机

vagrant box list

进入虚拟机

vagrant ssh

更多有用命令

vagrant -h
