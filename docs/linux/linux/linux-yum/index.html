常见的 yum 使用

  1. 添加源

例如 docker ce

wget https://download.docker.com/linux/centos/docker-ce.repo
mv docker-ce.repo /ect/yum.repo.d/

  1. 显示软件的可选版本

例如显示出 docker 的各个版本

yum list docker-ce –showduplicates | sort -r
