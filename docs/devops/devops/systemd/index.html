一个注册 ngrok （内网穿透工具） 的实例

  1. 编写ngrok.servcie 服务文件
  1. 拷贝服务文件到系统指定目录

sudo cp ngrok.service /etc/systemd/system

  1. 加载到系统中

sudo systemctl daemon-reload

  1. 启动服务

sudo systemctl start ngrok.service
