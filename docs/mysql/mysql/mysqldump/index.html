数据库导出

mysqldump \
–column-statistics=0 \
–user=root \
–password=123456 \
–compatible=postgresql \
databasename

PS 我用的 Mac Mysql workbench 带的 mysqldump 命令

/Applications/MySQLWorkbench.app/Contents/MacOS/mysqldump \
–column-statistics=0 \
–user=root \
–password=123456 \
–compatible=postgresql \

查询结果导出

mysql> SELECT * FROM runoob_tbl 
    -> INTO OUTFILE '/tmp/runoob.txt';
